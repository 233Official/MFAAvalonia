## ⚡ 技术栈速览

- **框架**: .NET 10 + Avalonia 11.3
- **模式**: MVVM (CommunityToolkit.Mvvm)
- **核心**: MaaFramework.Binding
- **UI 库**: SukiUI
- **日志**: Serilog
- **序列化**: Newtonsoft.Json
